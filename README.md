# video-storyboard-bot

Telegram-бот для раскадровки видео. Принимает видео или ссылку на ролик, автоматически находит ключевые кадры (смены сцен и крупные движения) и присылает их покадровой нарезкой в формате jpg.

## Возможности

- Приём видео **файлом** или **как видео** (до 20 MB — лимит Telegram).
- Приём по **ссылке** (Instagram, YouTube, TikTok и др.) — бот сам скачивает ролик через yt-dlp, без лимита 20 MB.
- **Автоматическая раскадровка всего ролика** — указывать время вручную не нужно.
- **Умный выбор ключевых кадров**: бот анализирует, насколько каждый кадр отличается от предыдущего, и отправляет до 18 самых заметных моментов (смены ракурса и крупные движения).
- Кадры приходят **альбомами** (документами, с маленьким превью), с паузой и автоповтором при лимите Telegram (ошибка 429).

## Как пользоваться

Команды писать не нужно — просто пришли боту:

- **ссылку** на ролик (начинается с `http`), или
- **видео** файлом / как видео.

Бот сразу обработает ролик и пришлёт ключевые кадры. Можно тут же отправлять следующий ролик.

Команда `/start` — приветствие и краткая справка.

## Установка и запуск

1. Установи Python 3.11 или 3.12 (на очень свежих версиях некоторые библиотеки могут не собраться).

2. Установи зависимости:

   ```bash
   pip install pyTelegramBotAPI opencv-python yt-dlp
   ```

   yt-dlp использует **FFmpeg**. Обычно ставится автоматически; если будет ошибка — установи отдельно:
   - Windows: `winget install ffmpeg`
   - macOS: `brew install ffmpeg`
   - Linux: `sudo apt install ffmpeg`

3. Получи токен бота у **@BotFather** в Telegram (`/newbot`).

4. В `main.py` вставь свой токен вместо `ВСТАВЬ_СЮДА_СВОЙ_ТОКЕН`:

   ```python
   bot = telebot.TeleBot('твой_токен')
   ```

5. Запусти:

   ```bash
   python main.py
   ```

   Бот работает, пока запущен скрипт. Остановить — `Ctrl+C`.

## Настройка

В функции `storyboard` можно поменять одну переменную:

- `MAX_FRAMES` — максимальное число отправляемых кадров (по умолчанию 18). Больше = подробнее, меньше = только самые яркие моменты.

## Технологии

- **pyTelegramBotAPI** — работа с Telegram Bot API.
- **OpenCV (cv2)** — анализ кадров и детекция смены сцен.
- **yt-dlp** — скачивание роликов по ссылке (Instagram, YouTube, TikTok и др.).

## Примечание о приватных видео

Публичные ролики обычно скачиваются без проблем. Для приватных видео Instagram может потребоваться авторизация — тогда в `ydl_opts` нужно добавить cookies из браузера, например:

```python
'cookiesfrombrowser': ('chrome',),
```
